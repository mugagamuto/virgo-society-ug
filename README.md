Super clean, mobile-first website starter for Virgo Society Uganda.

## Pages included
- `/` Home
- `/about`
- `/programs`
- `/loans`
- `/impact`
- `/donate`
- `/blog` and `/blog/[slug]`
- `/contact`
- `/legal/privacy` and `/legal/terms`
- Custom `not-found`

## Tech
- Next.js App Router (TypeScript)
- TailwindCSS
- Framer Motion (subtle animations)
- Lucide icons

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Customize quickly
- Site name/contact: `content/site.ts`
- Home metrics + hero text: `app/page.tsx`
- Blog sample posts: `content/blog/posts.ts`
